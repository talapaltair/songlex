# SongLex Grade 11 Website

Static website for the dissertation practical product:
"Enhancing High School Students' Lexical Competence through Song Materials."

## Open locally

Open `index.html` in a browser. No build step, package install, or local server is required.

## Files

- `index.html` - page structure and static sections
- `styles.css` - responsive layout and visual design
- `script.js` - unit data, rendered song modules, accordions, tabs, progress tracking, and placeholder form behavior
- `assets/songlex-hero.svg` - local visual asset for the hero section

## Copyright note

The website uses placeholders for official YouTube, Spotify, or official audio embeds. Do not paste full copyrighted lyrics into the website. Use target vocabulary, collocations, short phrases, paraphrase, and teacher-created tasks.
